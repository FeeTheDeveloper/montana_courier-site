# Montana Medical Courier — Website

Professional HIPAA-compliant medical courier services across Montana.

Built with **Next.js 14** (App Router), **TypeScript**, and **Tailwind CSS**. Deployed on **Vercel**.

## Quick Start

The Next.js application lives in the `montana-medical-courier/` directory. See [montana-medical-courier/README.md](montana-medical-courier/README.md) for full setup, environment variables, and deployment instructions.
